# EduNote 기획서

> 영어 학원 강사용 학생 관리 + 학부모 대시보드

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | EduNote |
| 대상 사용자 | 영어 학원 강사 (관리자), 학부모 (뷰어) |
| 핵심 목적 | 수업·시험 관리 자동화 + 학부모에게 학습 현황 투명 공유 |

---

## 2. 사용자 역할

### 강사 (Admin)
- 수업 생성 및 관리
- 학생 등록 및 그룹 배정
- 시험 출제 및 채점
- 독해 유형별 분석

### 학부모 (Parent)
- 자녀 학습 현황 조회 (읽기 전용)
- 시험 결과 및 추이 확인
- 공유 링크로 접근 (별도 계정 불필요)

---

## 3. 핵심 기능

### 3-1. 수업 관리 (Class)
- 수업 생성 / 수정 / 삭제
- 수업명, 요일, 시간, 수강생 목록
- 수업별 학생 배정

### 3-2. 학생 관리 (Student)
- 학생 등록 / 수정 / 삭제
- 학생별 수업 이력 조회
- 학부모 공유 링크 발급

### 3-3. 문제 유형 (Question Type)
- 유형 목록 관리 (독해, 문법 등 강사가 직접 설정)
- 시험 문항에 유형 태깅
- 유형별 정답률 분석

### 3-4. 시험 관리 (Exam)
- 시험 생성 (수업 연결, 날짜, 문항 수 설정)
- 문항별 문제 유형 지정
- 시험지 미리보기

### 3-5. 채점 (Grading) ← 핵심
- 학생별 답안 입력 (빠른 채점 UI)
- 자동 점수 계산
- 문항별 오답 집계
- 문제 유형별 정답률 계산

### 3-6. 학부모 대시보드 (Parent View)
- 공유 토큰으로 접근 (`/share/[token]`)
- 자녀 시험 점수 추이 차트 (Recharts)
- 문제 유형별 강약점 분석
- 최근 시험 결과 상세

### 3-7. 학생 이력 (History)
- 기간별 성적 추이
- 수업별 출석/시험 이력
- 전체 통계 요약

## 4. 기술 스택

| 분류 | 기술 |
|------|------|
| 프레임워크 | Next.js 16 (App Router, Turbopack) |
| UI | shadcn/ui + Tailwind CSS v4 |
| 상태관리 | TanStack Query + Zustand |
| 차트 | Recharts |
| 백엔드/DB | Supabase (PostgreSQL + Auth + RLS) |
| 아이콘 | lucide-react |
| 알림 | sonner (toast) |

---

## 5. DB 스키마 (주요 테이블)

```
teacher          — 강사 계정 (auth 연결)
class            — 수업
student          — 학생
class_student    — 수업-학생 연결 (다대다)
question_type    — 문제 유형 (독해, 문법 등)
exam             — 시험
exam_question    — 시험 문항 (유형 태깅)
grade            — 채점 결과 (학생 x 시험)
grade_detail     — 문항별 정오답
share_token      — 학부모 공유 링크
```

---

## 6. 페이지 구조

```
/login                      — 로그인
/signup                     — 회원가입

/dashboard                  — 수업 목록 (메인)
/dashboard/[classId]        — 수업 상세 (학생 목록, 시험 목록)

/students                   — 전체 학생 목록
/students/[studentId]       — 학생 상세 + 이력

/exams/[examId]             — 시험 상세
/exams/[examId]/grade       — 채점 화면

/settings/question-types    — 문제 유형 관리

/share/[token]              — 학부모 대시보드 (공개)
```

## 8. 비기능 요구사항

- 모바일 대응 (반응형)
- 채점 화면은 빠른 입력 최적화 (키보드 탐색)
- 학부모 링크는 로그인 불필요 (토큰 기반)
- RLS로 강사별 데이터 격리
